## WrappedStory Schema

Current payloads use `WrappedStoryV3`, a compact MessagePack schema documented in [WRAPPED_V3_SPEC.md](WRAPPED_V3_SPEC.md). It includes aggregate counts, monthly arrays, heatmaps, distributions, top short project names, co-occurrence edges, timeline events, session fingerprints, streaks, and token/model aggregates.

Legacy proposal examples below use a smaller `WrappedStory` shape for illustration only:

```
WrappedStory {
  y: 2025                          в”Җв”Җв”Җ Year
  n: "Adewale"                     в”Җв”Җв”Җ Display name (optional)

  p: 4                             в”Җв”җ
  s: 70                             в”Ӯ  Core counts
  m: 5316                           в”Ӯ  (aggregates only,
  h: 312                           в”Җв”ҳ  never content)

  t: ["Agent-driven", ...]         в”Җв”Җв”Җ Personality traits
  c: "Heavy delegation"            в”Җв”Җв”Җ Collaboration style
  w: "Steady flow"                 в”Җв”Җв”Җ Work pace

  pp: "Keyboardia"                 в”Җв”җ
  pm: 1873                          в”Ӯ  Highlights
  ci: 3                             в”Ӯ
  ls: 50.5                         в”Җв”ҳ

  a: [12,45,89,120,...]            в”Җв”Җв”Җ Monthly activity (12 values)

  tp: [{n,m,d}, ...]               в”Җв”Җв”Җ Top 3 projects
}
```
